# 同舟供应链中台一期 PRD

## 1. 项目背景

同舟供应链中台用于沉淀产品开发、库存、销售、物流、定价等供应链经营数据。前期以网页端为主，后续同步适配 PC、移动 H5、微信小程序。简道云作为人工维护数据基座，海外仓 WMS 作为库存与出库销售数据来源。

一期聚焦四个能力：

- 库存同步：按国家、仓库、SKU 同步实时可售、在途、锁定、异常库存。
- 订单日报：汇总各仓每日出库订单、销售件数、销售额、缺货/取消/异常。
- 动销监控：基于库存、近 7/14/30 天销量和安全库存天数，识别补货、滞销和断货风险。
- 对外产品库：展示直营/分销产品的国家、售价、库存、状态与基础资料。

## 3. 一期目标

### 3.1 业务目标

- 管理者每天能快速看到各国家库存、出库、动销与风险状态。
- 运营能发现需要备货、断货、滞销和库存数据异常的产品。
- 对外协作方能通过产品库查看分销/直营产品价格、库存与上架状态。
- 技术上先建立多 WMS、多仓库、多国家的接入模型，避免后续重复改造。

### 3.2 非目标

- 不在一期实现完整产品开发里程碑流程录入。
- 不在一期实现物流运费分摊、产品单价加权平均、资质办理等复杂核算。
- 不在一期直接替代简道云的人工录入能力。
- 不在一期实现小程序原生端，只预留响应式页面与 API 结构。

## 4. 用户角色

- 供应链管理员：配置仓库授权、查看同步状态、处理异常。
- 产品/采购人员：查看 SKU 库存、销量、补货预警。
- 运营人员：查看订单日报、国家维度销售表现。
- 分销/直营合作方：访问对外产品库，查看可售产品、价格和库存。

## 5. 核心功能

### 5.1 库存同步

- 支持按仓库配置 WMS 类型、接口地址、授权凭据、国家和币种。
- 支持手动触发同步和定时同步。
- 同步粒度：SKU、仓库、国家。
- 展示字段：总库存、可售库存、锁定库存、在途库存、近同步时间、同步状态。
- 异常处理：接口失败、SKU 未匹配、库存为负、同步延迟。

### 5.2 订单日报

- 按日期、国家、仓库生成出库日报。
- 展示订单数、销售件数、销售额、退款/取消、异常订单。
- 支持近 7 天趋势和国家贡献占比。
- 后续可扩展为导出 Excel、自动发送企业微信/飞书。

### 5.3 动销监控

- 计算近 7/14/30 天销量、日均销量和库存可售天数。
- 预警类型：
  - 断货：可售库存为 0 或低于 3 天销量。
  - 补货：库存可售天数低于安全库存天数。
  - 健康：库存可售天数在合理区间。
  - 滞销：近 30 天销量低且库存较高。
- 支持按国家、仓库、品类、负责人筛选。

### 5.4 对外产品库

- 展示直营/分销产品的产品图、SKU、国家、币种、售价、库存、状态。
- 支持搜索、国家筛选、销售模式筛选。
- 对外页面默认隐藏内部成本、供应商、授权凭据等敏感信息。
- 后续可增加访问口令、客户分组价、下载报价单。

## 7. 页面规划

- 总览 Dashboard：库存、订单、动销预警、同步健康度。
- 库存同步：仓库连接、同步任务、SKU 库存明细。
- 订单日报：日报指标、趋势、国家/仓库明细。
- 动销监控：预警列表、安全库存天数、补货建议。
- 产品库：内部管理视图和对外展示视图。
- 设置：WMS 授权、简道云表单映射、字段匹配。

## 8. 多端策略

- 第一阶段：网页端响应式布局，桌面为高密度驾驶舱，移动端为卡片和列表。
- 第二阶段：抽离 API 层和业务模型，为移动 H5 和微信小程序复用。
- 第三阶段：小程序端只承载轻量场景，如扫码查库存、产品库浏览、预警查看。

## 9. 接口接入原则

- 每个 WMS 供应商实现独立 adapter。
- 仓库授权与接口域名按仓库实例保存，不写死在代码中。
- 外部接口返回先落标准化中间层，再写入库存/订单业务表。
- 同步任务需要保存请求批次、耗时、成功数量、失败数量和错误摘要。
- 敏感凭据只保存在服务端或密钥管理服务，前端不暴露。

## 10. 验收标准

- 可以在网页端查看库存总览、订单日报、动销预警和产品库。
- 产品库支持桌面和移动端浏览。
- 仪表盘能清晰展示不同国家和仓库的库存与订单状态。
- 代码中已有 WMS adapter 和简道云数据源抽象，后续接真实 API 时无需重写页面。
- 对外产品库不展示内部敏感字段。

## 11. 后续路线
